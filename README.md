# Netcord-Client

#### 介绍
基于Python的局域网聊天应用
A LAN chat app powered by Python


#### 安装教程

点击右边的Release，推荐下载最新版本

#### 使用说明

1.  确保局域网内有一台电脑（或服务器，能存文件就行）
2.  将Netcord目录（服务器端，暂时还未开放使用，马上就有了）放到其中，并设置网络共享
3.  将服务器映射到本地磁盘（或是网络路径）
4.  打开Netcord，输入服务器地址以及账号密码
5.  聊天吧！
